# Variables

A variable is a box that can hold data. 

    hello = "Jeff"
    earth = "world"
    print(hello, earth, "\n")


    earth = hello
    print(hello, earth, "\n")

    hello = "car"
    print(hello, earth, "\n")




## Naming Conventions
    - No special characters.
    !yo
    - Can't start with a number.
    9cars
    8 trucks


    - underscores allowed
    front_door (snake case for Python)
    frontDoor  (camel case)

    - correct naming
    back_door
    back
    front23
