# Data Types

### Int
    Any whole number that doesn't have a decimal.
    231 3 51 1 2 
    -233324141
    44241
    -1 23 -1 32 1

### Float
    Any number that has a decimal point.
    27.0
    2.1
    -9.2

### String
    Anything surrounded by single or double quotation marks.
    A sequence of characters or numbers.
    "HEllo"
    "24.21"
    'Hey WHATS up'

### Bool
    True (1)
    False (0)
